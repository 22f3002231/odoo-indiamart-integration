IndiaMART Integration

Automatically Fetch IndiaMART Leads into Odoo CRM

This module provides a complete integration with the IndiaMART Pull API. It automates the process of fetching new business inquiries and creating them as qualified leads in your Odoo CRM, saving you time and ensuring no lead is missed.

Key Features

  • ✓ Automated lead fetching every 5 minutes.
  • ✓ Manual fetch option for specific date ranges.
  • ✓ Detailed mapping of all key lead data (Company, Address, Contact Info).
  • ✓ Prevents duplicate leads using IndiaMART's Unique ID.
  • ✓ Creates entries as "Leads" for proper sales qualification.
  • ✓ Built-in API Log Viewer to monitor all integration activity.

How to Use

Configuration

1. Go to the IndiaMART -> Configuration menu.
2. Enter your IndiaMART Pull API Key and save.
3. Use the "Test Connection" button to ensure your key is valid.

Usage

The system will automatically fetch new leads every 5 minutes. To fetch leads from a past period, use the IndiaMART -> Fetch Leads menu. To see a history of all API calls, visit the IndiaMART -> API Logs menu.
